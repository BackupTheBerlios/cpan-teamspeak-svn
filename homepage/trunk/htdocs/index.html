This is the Homepage of the Perl CPAN-Module Teamspeak.
It is a Interface for Perl-Programmers to the commercial Voice-Server.
At first you have to create a Teamspeak-Handle of the Type Telnet, Web or SQL. Next you have to connect to the Teamspeak-Program.

It depends of the type, what you can do with your Handle, but you will get the same Objects:
  • Player
  • Server
  • Channel
  • Ban
  • Privileges
The Interface is different (depends of the Teamspeak-Handle-Type) and documented by perldoc Teamspeak::Ban for example. But you can view the Objects and Methods also as a FreeMind-Mindmap:
  • Here is a HTML-Export with Javascript.
  • This is the FreeMind-Source.
